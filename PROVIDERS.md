# AI API Guide

This document provides instructions on how to obtain API keys for the providers supported in **PixelForge**.

## 1. Pollinations.ai (Free)
- **Website**: [pollinations.ai](https://pollinations.ai/)
- **Cost**: Free
- **API Key**: Not required.
- **Usage**: Unlimited generation (subject to fair use).

## 2. Hugging Face
- **Website**: [huggingface.co](https://huggingface.co/)
- **Cost**: Free tier available (rate limited), with paid PRO options.
- **How to get Key**:
    1. Sign up/Log in to Hugging Face.
    2. Go to **Settings** > **Access Tokens**.
    3. Click **Create New Token**.
    4. Set permissions (Read access is sufficient for Inference).
    5. Copy the token (starts with `hf_...`).

## 3. PixelLab.ai
- **Website**: [pixellab.ai](https://pixellab.ai/)
- **Cost**: Paid (credits based).
- **How to get Key**:
    1. Sign up/Log in at PixelLab.ai.
    2. Go to your **Account Dashboard** or **Developer Settings**.
    3. Find the **API Keys** section.
    4. Generate a new API Key.

## 4. Recraft (RareCraft)
- **Website**: [recraft.ai](https://recraft.ai/)
- **Cost**: Paid (credits based).
- **How to get Key**:
    1. Sign up/Log in at Recraft.ai.
    2. Navigate to your **Profile** or **Usage** page.
    3. Look for the **API** section.
    4. Generate a new API Key.
